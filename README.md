# Pencil-Sketch-python
used to convert an image to pencil sketch

PACKAGES REQUIRED:
pip3 install numpy,
install OpenCV Python (not a command line)

NOTE:
Have the image to be pencil sketched inside python directory or specify its path in cv2.imread('')
